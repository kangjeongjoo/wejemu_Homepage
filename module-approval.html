KEY FUNCTIONS

전자결재 핵심 기능

복잡한 결재 프로세스를 디지털화하여 업무 속도와 투명성을 높입니다.

📄

다양한 결재 양식

인사발령·급여대장·지출결의·자금이체 등 업무별 결재 양식을 제공합니다. 기업 특성에 맞게 양식을 커스터마이징할 수 있습니다.

인사양식회계양식자금양식
🔗

유연한 결재선 설정

직급·부서 기반 결재선을 자동 구성하고, 문서 유형·금액에 따라 결재선을 다르게 설정할 수 있습니다. 결재 단계와 결재자를 자유롭게 조합합니다.

자동구성조건별설정금액기준
📱

모바일 결재

스마트폰에서 결재 서류를 검토하고 즉시 승인·반려·의견 추가가 가능합니다. 외근·출장 중에도 결재 업무가 중단되지 않습니다.

모바일앱즉시처리외근결재
🔁

위임·대리 결재

출장·휴가 등 부재 시 결재를 지정된 대리인에게 위임할 수 있습니다. 위임 기간을 설정하면 자동으로 대리 결재자에게 요청이 전달됩니다.

위임설정기간지정대리결재
⚡

병렬·동시 결재

여러 결재자가 동시에 검토하는 병렬 결재를 지원합니다. 급여 지급처럼 시간이 촉박한 업무의 결재 속도를 획기적으로 단축합니다.

동시결재병렬처리결재단축
🔔

알림·이력 추적

결재 요청·처리 시 담당자에게 실시간 알림을 발송합니다. 모든 결재 이력이 자동으로 기록되어 감사와 내부통제에 활용할 수 있습니다.

실시간알림이력조회감사지원

결재와 실무가 자동으로 연결됩니다

전자결재 승인이 완료되면 회계 전표 생성, 자금 이체 지시, 급여 처리 등의 후속 업무가 자동으로 연동됩니다. 결재 완료 후 별도 처리 없이 업무가 진행됩니다.

지출결의 승인 → 회계 전표 자동 생성
자금이체 승인 → 뱅킹 이체 지시 자동 연동
급여대장 승인 → 급여 이체 처리 자동 연동
